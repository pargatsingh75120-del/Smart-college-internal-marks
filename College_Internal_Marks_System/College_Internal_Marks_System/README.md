# 🎓 College Internal Marks System

An advanced AI-powered College Internal Marks Management System developed using HTML, CSS, JavaScript, Excel integration, Chart.js, and ML-ready features.

## 🚀 Features
- Teacher & Student Login
- Excel Upload/Export
- Internal Marks Calculation
- Student Performance Card
- PDF Download
- Chart Analytics
- ML Integration for Colab
- Responsive UI
- Data Visualization

## 📂 Project Structure
- src/ → Source code
- assets/ → Images & icons
- charts/ → Analytics modules
- auth/ → Authentication logic
- export/ → PDF & Excel export
- ml/ → Machine Learning helper scripts
- docs/ → Documentation
- tests/ → Testing files
- api/ → API placeholders

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- Chart.js
- SheetJS
- jsPDF
- html2canvas
